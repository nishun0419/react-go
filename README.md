# react-go

## 概要
reactとgolangの開発環境です。

## 開発環境
フロントエンド react  
バックエンド golang   
データベース mysql  

## 待機ポート番号
react 3000  
golang 8080,8888(テスト用)  
mysql 3306

## URL
react(フロント) [http://localhost:3000/]()  
golang(バック) [http://localhost:8080/]()

## DB情報
ユーザー: root  
パスワード: password  
DB名: sample  
ホスト: mysql
